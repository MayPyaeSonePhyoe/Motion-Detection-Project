Overview :

The Motion Detection System is a computer vision project that detects movement in real-time using a camera. The system continuously monitors video frames and identifies changes between them to determine whether motion has occurred.

This project demonstrates how motion detection can be implemented using image processing techniques and basic computer vision algorithms. The system can be useful for simple surveillance and monitoring applications.

Features :

Real-time video capture from a camera

Detection of movement in video frames

Highlighting moving objects using bounding boxes

Continuous monitoring of the environment

Simple and efficient motion detection algorithm

Technologies Used :

This project was developed using the following tools and technologies:

Python – main programming language

OpenCV – used for image processing and motion detection

Visual Studio Code – development environment
